 Having a positive social interaction thought the day is the best thing that can happen to a person. 
 Well, I did that for the people. An app (Android app) that uses Kotin code to 
 generate the needs for the user in order tell them what to do for the day within a specific time.
 The application allows the user to input/enter a specific time in a day which are (
 Morning, Afternoon …), that input is then processed within a code then display 
 suggestions on what to do for the day based on the time you entered. Basically, the 
 purpose of I creating this app is provide people with meaningful suggestions based
 on different times of the day which can strengthen relationships between loved ones,
including friends or might be a person who you just met but end up having a tight bond
which is referred to as a loved one.

There are some other reasons on why I did this application. The reasons are that I want
to show people the use of logic in Kotlin language code. This elevates the knowledge of 
Koltin language and show people how fun is it to use this language. The app itself has a 
user input were time (Morning, etc.) is entered using an EditText. After entering the time
of the day, the app starts processing the value that was entered and uses a loop called
an IF loop to check whether if the input matches the values in coded within the loop. 
The app then finds the correct value that matches the input then displays a relevant 
message using a TextView. If whether the user entered a wrong time on the input box an 
error message is then displayed, if that’s not the case there’s a clear button which
clear’s all fields if the user typed the wrong text.
